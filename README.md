# front-end-ts-vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build-prod
```

### Compiles and minifies for uat
```
npm run build-uat
```

### Compiles and minifies for sit
```
npm run build-sit
```


### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```

### 启动mock服务器，所有请求会被代理到本地的mock服务器
```
npm run mock
```

### Compiles and minifies for production
```
npm run build
```
